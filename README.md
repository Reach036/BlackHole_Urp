# BlackHole_Urp
A blackhole shader in Unity Universial Render Pipeline(Unity6.2, older version may also available with some modification, but TAA and bloom is recommended)  
key feature: Kerr blackhole, ray-marching, volumetric acceleration disk, blue and red-shift, angular momentum effect  
not impled depth buffer-ray intersect, so intersect with other objects is not supported!  
skybox reading disabled by comments.
